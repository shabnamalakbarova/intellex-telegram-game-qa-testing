# API Testing Notes

## Tools Used
- Postman
- Chrome DevTools

## Checked Status Codes
- 200 OK
- 400 Bad Request
- 404 Not Found
- 502 Bad Gateway

## API Validation Performed
- Response status validation
- Response body structure validation
- Error handling checks
- Invalid input testing

## Observations
- Some actions return 404 errors
- Some requests return repeated 400 responses
- Server instability may cause 502 errors
- Error messages are not always user-friendly
