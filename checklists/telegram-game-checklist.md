# Telegram Game QA Checklist

## Launch

- [ ] Game opens from Telegram successfully
- [ ] Loading screen works correctly
- [ ] No black screen appears
- [ ] Main page loads without errors

## Duel

- [ ] Duel section opens
- [ ] Start/join duel works
- [ ] Answer selection works
- [ ] Duel result is displayed

## Gift Market

- [ ] Gift market opens
- [ ] Gift icons match gift names
- [ ] Gift purchase button works
- [ ] Balance updates after purchase

## Bonus

- [ ] Bonus activation button works
- [ ] Bonus balance updates correctly
- [ ] Error messages are clear

## Chat

- [ ] Message can be sent
- [ ] Message is received by other user
- [ ] Sending error is displayed if request fails
