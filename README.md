# 🎮 Intellex Telegram Game QA Testing Project

## 📌 Project Overview
This project represents QA testing performed on a Telegram-based game platform.

All materials are anonymized and prepared for portfolio purposes.

---

## 🔍 Testing Scope

- Functional Testing
- UI Testing
- API Testing
- Telegram Web App Testing
- Exploratory Testing
- Bug Reporting

---

## 🧪 Tested Features

- Telegram Web App launch
- Duel system
- Chat functionality
- Bonus activation
- Gift market
- Leaderboard
- Payment / coin flow
- User profile

---

## 🐞 Key Bugs Identified

- Telegram mini app opens with black screen
- Buttons clickable but non-functional
- Navigation redirects incorrectly
- Bonus activation returns 404
- Gift icons mismatch item names
- Chat messages not received
- Repeated 400 / 404 / 502 responses

---

## 📊 Testing Approach

- Exploratory testing
- API response validation
- DevTools network analysis
- Telegram Web App testing
- UI & usability validation
